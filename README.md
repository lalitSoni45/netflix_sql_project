## 📖 Overview

This project analyzes the Netflix Titles dataset using PostgreSQL. It covers data exploration, aggregation, text processing, regular expressions, CTEs, window functions, and business-oriented SQL questions.

## ❓ Business Questions Solved

1. Count Movies vs TV Shows
2. Find the Most Common Rating for Movies and TV Shows
3. List All Movies Released in 2020
4. Find the Top 5 Countries with the Most Content
5. Find the Longest Movie and TV Show
6. Find Content Added in the Last 5 Years
7. Find Content Directed by Rajiv Chilaka
8. List TV Shows with More Than 5 Seasons
9. Count Content Items in Each Genre
10. Top 5 Years with Highest Content Release in India
11. Find Documentary Content
12. Find Content Without a Director
13. Find Salman Khan Movies Released in the Last 10 Years
14. Top 10 Actors Appearing in Indian Movies
15. Categorize Content as Good or Bad Based on Description
