# Netflix SQL Data Analysis Project

## Overview
This project contains SQL analysis performed on the Netflix Titles dataset using PostgreSQL.

## Files

📊 Dataset:
- [netflix_titles.csv](./netflix_titles.csv)

📝 SQL Solutions:
- [solutions.sql](./solutions.sql)
  
### netflix_titles.csv
Dataset containing Netflix movies and TV shows information.

### solutions.sql
Contains solutions for 15 SQL business questions.

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- CASE WHEN
- CTE
- Window Functions
- STRING_TO_ARRAY()
- UNNEST()
- Regular Expressions
- Aggregate Functions

## Dataset Information

- Dataset Name: netflix_titles.csv
- Database: PostgreSQL

## Questions Solved

1. Count Movies vs TV Shows
2. Most Common Rating
3. Movies Released in 2020
4. Top 5 Countries with Most Content
5. Longest Movie and TV Show
6. Content Added in Last 5 Years
7. Content Directed by Rajiv Chilaka
8. TV Shows with More Than 5 Seasons
9. Count Content Items in Each Genre
10. Top Years with Highest Content Release in India
11. Find Documentary Content
12. Content Without Director
13. Salman Khan Movies in Last 10 Years
14. Top 10 Actors in Indian Movies
15. Categorize Content as Good or Bad
